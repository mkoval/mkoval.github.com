OpenRAVE Software

  • prpy - Python utilities used by the Personal Robotics Laboratory

    PrPy is a Python library used by the Personal Robotics Laboratory at Carnegie Mellon University. This library includes robot-agnostic utilities that make it easier to use OpenRAVE in Python scripts. This includes a high-level planning pipeline, helper functions, and visualization tools.

  • or_ompl - OpenRAVE bindings for OMPL

    OpenRAVE bindings for the OMPL suite of motion planning algorithms. This package provides an OpenRAVE planning plugin that delegates planning requests to an OMPL planner or path simplifier.

  • or_fcl - OpenRAVE bindings for FCL

    OpenRAVE bindings for the Flexible Collision Library (FCL). This package provides an OpenRAVE collision checking plugin that uses FCL to perform collision checks. Benchmarks show that or_fcl is 2–7× faster than the collision detectors included with OpenRAVE.

  • or_rviz - OpenRAVE bindings for RViz

    OpenRAVE bindings for RViz. This package provides two OpenRAVE viewer plugins that publish an OpenRAVE environment as interactive markers that may be visualized in RViz.

  • or_urdf - OpenRAVE loader for URDF and SRDF files

    OpenRAVE plugin for loading a URDF and SRDF files into OpenRAVE.

  • or_parabolicsmoother - OpenRAVE plugin for smoothing trajectories

    OpenRAVE plugin for smoothing robot manipulator trajectories under geometric constraints and bounded acceleration. This plugin is based on code originally written by Kris Hauser and distributed with the paper "Fast Smoothing of Manipulator Trajectories using Optimal Bounded-Acceleration Shortcuts."

Hardware Drivers

  • stargazer - Hagisonic Stargazer Driver

    ROS driver for the Hagisonic Stargazer. This package provides a Python interface for communicating to the Stargazer over RS-232 and a ROS node that publishes all detected markers as ROS messages.

  • fieldforce_tcm - PNI Fieldforce TCM Digital Compass Driver

    ROS hardware driver for the PNI Fieldforce TCM digital compass. Warning: This code is undocumented and has not been updated for Catkin.

  • stereo_webcam - USB Webcam Driver for Stereo Vision

    ROS hardware driver that capture images from hardware-synchronized PS3 Eye cameras using direct Video4Linux system calls. This package is intended to be used to convert two PS3 Eye cameras into an inexpensive stereo pair. Warning: This code is undocumented and has not been updated for Catkin.

  • jaguar - Jaguar Motor Control Driver

    ROS hardare driver for controlling the TI Jaguar motor controller. This package communicates with one Jaguar over RS-232 and uses its built-in CAN bus bridge to control additional daisy-chained hardware. Warning: This code is undocumented and has not been updated for Catkin.

  • hax - Hardware Abstraction for Vex

    Hardware Abstraction for VEX (HAX) is a C hardware abstraction layer for the VEX Robotics PIC18 and ARM Cortex M3 micrcontrollers. HAX provides the first open source development pipeline for VEX robotics.
