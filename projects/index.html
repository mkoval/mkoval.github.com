  • Rutgers Navigator, Intelligent Ground Vehicle Competition

    The 19th Annual Intelligent Ground Vehicle Competition (IGVC) marked Rutgers University's rookie year of competing in IGVC and the completion of the Navigator. As vision lead, I was responsible for developing the Navigator's custom trinocular stereo vision, obstacle detection, and lane tracking systems. For the upcoming 20th IGVC I will be serving as team lead.

  • Policy Search for Control of a Quadrotor Helicopter

    Policy search is a branch of reinforcement learning that has yielded promising results for problems with a high-dimensional state-action space. This is especially true in robotics, where sensor readings and actuator controls are both continuous. Over the period of a year, I researched the application of policy search to the control of an autonomous quadrotor helicopter with the Rutgers Laborator for Real-life Reinforcement Learning (RL3) and advisor Dr. Michael Littman.

  • NetBots: Swarm Robotics Target Recognition

    NetBots is a novel swarm robotics algorithm to improve detection accuracy in the distributed automatic target recognition (ATR) problems. Prior work (Dasgupta et al. and Martinoli et al.) have applied swarm robotics to the ATR problem by having a large number of cheap, inaccurate robots "vote" on whether a given object is a target.

  • Hardware Abstraction for Vex (HAX)

    HAX is a hardware abstraction layer that provides a common interface to the Vex PIC Microcontroller and Vex Cortex Microcontroller. Any code that interfaces with hardware through HAX can seamlessly run on the PIC and ARM architectures with minimal changes.
